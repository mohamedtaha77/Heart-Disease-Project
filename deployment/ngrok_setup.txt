Ngrok Deployment Instructions
=============================

1. Download Ngrok:
   Visit https://ngrok.com/download and download for your OS.
   Extract and place the executable in a directory accessible from the terminal.

2. Authenticate Ngrok:
   Run this once to link your account:
   ngrok config add-authtoken YOUR_AUTHTOKEN

3. Run the Streamlit app:
   In the 'ui/' folder, start the app:
   streamlit run app.py

4. Start Ngrok:
   In a new terminal:
   ngrok http 8501

5. Copy the generated URL (e.g., https://xxxx.ngrok-free.app) and share it.

Note:
- The app remains live as long as both terminals (Streamlit + Ngrok) are running.
